# ACS Data for Indianapolis

Census American Community Survey (ACS) 5-year estimates by tract.

| Table    | Description                  | DiD Role  |
|----------|------------------------------|-----------|
| B01003   | Total Population             | Control   |
| B19013   | Median Household Income      | Control   |
| B25001   | Housing Units                | Control   |
| B25070   | Gross Rent as % of Income    | Outcome   |

Files: `ACSDT5Y{year}.{table}-Data.csv` (e.g. ACSDT5Y2016.B25070-Data.csv)
